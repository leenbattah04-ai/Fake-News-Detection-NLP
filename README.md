# 📰 Fake News Detection using NLP

## 📌 Project Overview
This project uses Natural Language Processing (NLP) and Machine Learning to classify news articles as real or fake.

---

## 📊 Dataset
The dataset is publicly available on Kaggle and can be downloaded using the following search:
Fake News Detection Dataset (Kaggle)

⚠️ Note:
The dataset is not included in this repository due to size limitations.

---

## ⚙️ Workflow
- Data loading
- Handling missing values
- Text preprocessing
- Stopwords removal
- Stemming
- TF-IDF vectorization
- Logistic Regression model training

---

## 🧠 Technologies Used
- Python
- Pandas
- NumPy
- NLTK
- Scikit-learn

---

## 📈 Output
The model predicts:
- 0 → Real News
- 1 → Fake News
